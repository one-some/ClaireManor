# The Manor Claire
CSC 1313 Homework

## Usage
You will need the `raylib` package:
`pip install -r requirements.txt`

## Features
- Basic layout engine
- Text processing (tiny tag-based markup for styling, templating for natural-ish sounding language)
- HiDPI Support
- Dumb battle AI
- Cool async architecture
